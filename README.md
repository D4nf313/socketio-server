# socketio-server
Este es el servidor Socket.IO para usar este servidor HTTP (socketIo(server)). para implementarlo con la app viajesApp

clona este proyecto con git clone 
abre el archivo socketio-server con una terminal y teniendo la version de node 17 o superior
ejecuta npm i 
al instalar las dependencias corectamente escribe el comando node server.js y debe correr el servidor en el puerto 3000
Asegurate de tener corriendo este servidor antes de la aplicacion viajesApp
